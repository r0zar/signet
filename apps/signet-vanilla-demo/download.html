Installation Guide

Since Signet is currently in development, it needs to be installed manually as an unpacked extension. Follow these steps to get started:

Download the Extension

Click the download button above to get the latest version of the Signet extension (.zip file).

Extract the Files

Locate the downloaded ZIP file and extract its contents to a folder on your computer. Make sure to remember where you extracted the files.

Right-click → Extract All... → Choose a destination → Extract

Open Chrome Extensions Page

Open Google Chrome and navigate to the extensions page by entering the following in your address bar:

chrome://extensions/

You can also access this page by clicking the puzzle piece icon in the top-right corner of Chrome, then clicking "Manage Extensions" at the bottom of the dropdown.

Enable Developer Mode

In the top-right corner of the extensions page, toggle on "Developer mode".

This will reveal additional developer options including the "Load unpacked" button.

Load Unpacked Extension

Click the "Load unpacked" button that appears after enabling Developer Mode.

In the file browser that opens, navigate to the folder where you extracted the Signet extension files (should contain files like manifest.json), and click "Select Folder".

Pin the Extension (Optional)

To easily access Signet, click the puzzle piece icon in Chrome's toolbar, find Signet in the list, and click the pin icon to keep it visible in your toolbar.

Important Note

Since this extension is in development and loaded in Developer Mode, Chrome may show a message about removing developer mode extensions when you restart your browser. This is normal for unpacked extensions. Simply click "Cancel" to keep the extension, or you may need to re-enable it after restarting.

Security Notice

Only install Chrome extensions from trusted sources. The Signet extension is designed to manage blockchain transactions and digital signatures, so it's important to verify you're installing the official version from the Charisma team.
